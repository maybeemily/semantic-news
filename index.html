What is DNS?

Domain Name System - or DNS is a decentralized naming system for things that are connected to the internet, associating information with domain names.

What happens when you type in a URL?

When you request a domain name, your computer asks the domain name resolver to query the root server, which then refers the resolver to the nameserver (.org nameserver, .com nameserver...) which is then referred to the domain nameserver. Finally the resolver will receive an authoritative answer which will be the IP address related to the requested URL. The resolver will cache recent lookups in case you search for it again, so it doesn't need to go on this journey every time.

But where does this happen?

This all happens behind the scenes in your web browser, email client, video game player, etc.
